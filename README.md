# Sistema de Agendamento – Conj. Res. Port de France

Projeto desenvolvido no âmbito do Projeto Integrador da UNIVESP.

## Objetivo
Desenvolver um sistema web para agendamento de espaços coletivos do condomínio, substituindo o processo manual.

## Funcionalidades
- Identificação do morador (nome e apartamento)
- Cadastro de reservas
- Listagem de reservas
- Bloqueio de conflitos de horário
- Persistência de dados no navegador

## Tecnologias
- HTML
- CSS
- JavaScript
- Git/GitHub

## Validação
O sistema foi testado por usuários simulando moradores do condomínio, que realizaram reservas e forneceram feedback para melhorias.
